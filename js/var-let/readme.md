## this
 - this是什么？
   函数**执行**的时候，会立即生成的对象,叫this
   由调用方式决定，指向最后的调用者
   严格模式下，this指向undefined，没有必要的
   运行方式：
   - 一般运行
   - new 方式运行，this指向实例对象
   - 对象的方法 this对象
   - 事件处理函数来运行，指向事件对象
   - 箭头函数 本身没有this，其中的this其实是作用域链中的this
